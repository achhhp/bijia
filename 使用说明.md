# 供应商比价软件使用说明

## 软件功能

本软件用于对多个供应商的报价单进行比价分析，主要功能包括：

1. **多文件上传**：支持上传至少3份报价单（Excel或CSV格式）
2. **自动价格比较**：自动找出每个物料的最低价和对应供应商
3. **供应商中标统计**：统计每个供应商中了哪些低价物料
4. **结果展示**：直观展示分析结果，包括最低价分析和供应商中标统计
5. **报告导出**：支持将分析结果导出为Excel格式的报告

## 系统要求

- Python 3.6 及以上版本
- 所需依赖包：
  - pandas
  - openpyxl（用于Excel文件处理）
  - tkinter（Python标准库，用于GUI界面）

## 安装方法

1. 确保已安装Python 3.6 及以上版本
2. 安装所需依赖包：
   ```bash
   pip install pandas openpyxl
   ```
3. 下载 `vendor_price_comparison.py` 文件到本地

## 使用步骤

### 1. 准备报价单文件

报价单文件需要包含以下列：
- **物料**：物料名称
- **价格**：物料价格

支持的文件格式：
- Excel文件（.xlsx, .xls）
- CSV文件（.csv）

### 2. 运行软件

在命令行中执行：
```bash
python vendor_price_comparison.py
```

### 3. 上传报价单

1. 点击 "上传报价单" 按钮
2. 选择至少3份报价单文件
3. 软件会自动显示已上传的文件列表

### 4. 开始分析

1. 确保已上传至少3份报价单
2. 点击 "开始分析" 按钮
3. 软件会自动分析每个物料的最低价和对应供应商
4. 分析结果会显示在 "分析结果" 区域

### 5. 查看结果

分析结果分为两个标签页：

- **最低价分析**：显示每个物料的最低价和对应供应商
- **供应商中标统计**：显示每个供应商中标的物料数量和具体物料

### 6. 导出报告

1. 点击 "导出报告" 按钮
2. 选择导出路径和文件名
3. 软件会生成一个Excel格式的分析报告，包含两个工作表：
   - **最低价分析**：每个物料的最低价和对应供应商
   - **供应商中标统计**：每个供应商的中标情况

### 7. 清空数据

点击 "清空" 按钮可以清空所有上传的文件和分析结果，重新开始分析。

软件包中包含三个示例报价单文件：

- `供应商C.xlsx`

可以使用这些示例文件测试软件功能。

## 注意事项

1. 报价单文件必须包含 "物料" 和 "价格" 两列
2. "价格" 列必须包含数字值
3. 建议使用统一的物料名称格式，确保不同供应商的相同物料能够正确匹配
4. 软件会使用文件名作为供应商名称，建议使用清晰的文件名格式
5. 对于大型报价单文件，分析可能需要一定时间，请耐心等待

## 故障排除

### 常见错误及解决方案

1. **文件解析错误**
   - 检查文件格式是否正确
   - 确保文件包含 "物料" 和 "价格" 两列
   - 确保 "价格" 列包含有效的数字值

2. **分析结果为空**
   - 检查是否上传了足够的报价单文件（至少3份）
   - 检查报价单文件是否包含有效的数据

3. **导出失败**
   - 检查导出路径是否存在
   - 确保有足够的权限写入文件
   - 确保文件未被其他程序占用
